Tukang Hitung is a full-stack SaaS platform built to digitalize the construction project lifecycle, this solution to bridge the gap between financial estimation and onsite execution. Implemented a robust architecture to handle complex calculations for Rencana Anggaran Biaya (RAB), allowing for dynamic updates between planned and actual costs.
